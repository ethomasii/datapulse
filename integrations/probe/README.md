# Probe (optional)

Similar in *role* to [ServicePulse probe](https://github.com/servicepulsehq/integrations/tree/main/probe): optional small jobs that run **inside your network** (poll a private URL or DB) and then signal health outward—for example by ensuring a gateway host can reach the control plane or by complementing heartbeats.

This folder is reserved for future minimal probe examples; the primary integration surface remains the **gateway agent** container.
